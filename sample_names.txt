# sample_names.txt
# ================
# 카드해설 docx 작성 중 본문에 들어간 샘플 이름 목록.
# extract_cards.py와 make_backup_docx.py가 이 파일을 읽어
# 각 이름을 {NAME} 변수로 자동 치환한다.
#
# 사용법:
#   - 한 줄에 한 이름씩 적어둠
#   - '#'으로 시작하는 줄은 주석 (무시됨)
#   - 빈 줄은 무시됨
#   - 이름만 적으면 됨. "OOO님"은 자동으로 "{NAME}님"이 됨.
#
# ⚠️ 보안:
#   이 파일은 개인정보(샘플 이름)가 들어있으므로
#   GitHub에 올리지 말 것. .gitignore에 추가 권장.
#
# 예시 (실제 사용 시 아래 줄들을 본인 샘플 이름으로 교체):
# 홍길동
# 김철수
